
Тип bool: True или False

```Python
4 > 7  # False
4 > 1  # True

res = 4 > 7  # >>>print(res)  # True
```

### Операторы сравнения 
```Python
<  # Сравнение на меньше
>  # Сравнение на больше
<=  # Сравнение на меньше или равно
>=  # Сравнение на больше или равно
==  # Сравнение на равенство
!=  # Сравнение на неравенство
```

### Проверка на чётность и нечётность
```Python
x % 2 == 0 # На четность, если True

x % 2 != 0 
x % 2 == 1  # На нечётность, если True
```
